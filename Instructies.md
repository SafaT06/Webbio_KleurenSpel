# Instructies om het te runnen

## Lokaal draaien
Hieronder stappen die je kunt volgen om de applicatie te runnen. 
Doe dit via je terminal

1. Repo clonen
```bash
git clone https://github.com/SafaT06/Webbio_KleurenSpel.git
```

2. Juiste directory
```bash
cd kleuren-spel
```

3. Node modules
```bash
npm install
```

4. Start react 
```bash
npm start
```

Als het goed is moet nu de applicatie draaien automatisch op port 3000 -> localhost:3000
Je komt automatisch op de startscherm, succes met hoge scores te halen.
